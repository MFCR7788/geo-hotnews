> 作者：程序员鱼皮 | 项目地址：https://github.com/liyupi/yupi-hot-monitor

## 一、项目概览

### 1.1 核心定位
这是一个**AI 热点监控工具**，通过配置关键词，自动从多个信息源聚合抓取内容，利用 AI 进行真假识别和相关性分析，并通过 WebSocket 实时推送和邮件通知用户。

### 1.2 技术栈
| 层级 | 技术选型 |
|------|----------|
| **后端** | Express 5 + TypeScript + Prisma (SQLite) |
| **前端** | React 19 + Vite + TailwindCSS 4 + Framer Motion |
| **实时通信** | Socket.io (WebSocket) |
| **AI 能力** | OpenRouter API (DeepSeek V3.2) |
| **数据源** | 8+ 平台 (Twitter, Bing, HackerNews, 搜狗, B站, 微博等) |

## 三、核心模块分析

### 3.1 数据库设计 (Prisma Schema)

```prisma
// 关键词表
model Keyword {
  id        String    @id @default(uuid())
  text      String    @unique        // 关键词内容
  category  String?               // 分类
  isActive  Boolean   @default(true) // 是否激活
  createdAt DateTime  @default(now())
  updatedAt DateTime  @updatedAt
  hotspots  Hotspot[]              // 关联热点
}

// 热点表 (核心数据)
model Hotspot {
  id              String    @id @default(uuid())
  title           String                   // 标题
  content         String                   // 内容
  url             String                   // 原文链接
  source          String                   // 来源平台
  sourceId        String?                  // 原始ID

  // AI 分析结果
  isReal          Boolean   @default(true) // 是否真实
  relevance       Int       @default(0)    // 相关性 0-100
  relevanceReason String?                   // 分析理由
  keywordMentioned Boolean?                 // 是否直接提及关键词
  importance      String    @default("low")// low/medium/high/urgent
  summary         String?                   // AI 摘要

  // 互动数据
  viewCount       Int?
  likeCount       Int?
  retweetCount    Int?
  replyCount      Int?
  commentCount    Int?
  quoteCount      Int?
  danmakuCount    Int?

  // 作者信息
  authorName      String?
  authorUsername  String?
  authorAvatar    String?
  authorFollowers Int?
  authorVerified  Boolean?

  publishedAt     DateTime?
  createdAt       DateTime  @default(now())
  keywordId       String?
  keyword         Keyword?  @relation(...)

  @@unique([url, source])  // URL+来源 联合唯一索引
}

// 通知表
model Notification {
  id        String   @id @default(uuid())
  type      String   // hotspot/alert
  title     String
  content   String
  isRead    Boolean  @default(false)
  hotspotId String?
  createdAt DateTime @default(now())
}

// 设置表
model Setting {
  id    String @id @default(uuid())
  key   String @unique
  value String
}
```

  // 1. 获取所有激活的关键词
  const keywords = await prisma.keyword.findMany({ where: { isActive: true } });

  for (const keyword of keywords) {

    const accountResult = await detectAndFetchAccount(keyword.text);

        searchHackerNews(keyword.text),
        searchSogou(keyword.text),
        searchBilibili(keyword.text),
        searchWeibo(keyword.text)
      ]);

    // 5. 去重 → 过滤 → 排序

    // 6. AI 分析 + 阈值过滤
    const analysis = await analyzeContent(fullText, keyword.text, preMatch);
    if (!analysis.isReal || analysis.relevance < 50) continue;

    const hotspot = await prisma.hotspot.create({ data: {...} });

## 六、项目亮点分析

### 6.1 架构设计亮点
1. **多源聚合**: 8+ 数据源并行抓取，提高信息覆盖
2. **AI 驱动**: Query 扩展 + 真实性判断 + 相关性分析
3. **实时推送**: WebSocket 实现秒级通知
4. **定时任务**: 每 30 分钟自动检查，无需人工干预

### 6.2 工程化亮点
1. **TypeScript**: 全栈类型安全
2. **Prisma ORM**: 类型安全的数据库操作
3. **速率限制**: 每个数据源独立的请求间隔控制
4. **去重机制**: URL 标准化去重 + 数据库唯一索引
5. **容错处理**: Promise.allSettled 处理部分失败

### 6.3 可改进点
1. **数据源依赖**: 部分爬虫可能因反爬策略失效
2. **AI 成本**: 每次分析都调用 API，成本随数据量线性增长
3. **数据库**: SQLite 适合小规模，生产环境建议迁移 PostgreSQL
4. **缓存**: 扩展关键词结果可加入 Redis 缓存
