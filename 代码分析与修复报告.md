## 二、🔴 严重问题

### 1. 前端目录名包含空格
**位置**: `/client 2/`
**问题**: 目录名包含空格会导致命令行工具、脚本、构建系统出现问题。
**修复**: 重命名为 `client/`。

### 2. 手动触发热点扫描未加认证
**位置**: `server/src/index.ts:75`
```typescript
app.post('/api/check-hotspots', async (req, res) => {
  // 缺少 requireAuth，任何人都可以触发扫描
```
**风险**: 未认证用户可以无限触发热点扫描，消耗 API 配额和服务器资源。
**修复**: 添加 `requireAuth` 中间件。

### 3. JWT Secret 硬编码默认值
**位置**: `server/src/utils/jwt.ts:26,38,51`
```typescript
const secret = process.env.JWT_SECRET || 'mfcr-hotnews-jwt-secret-change-in-production';
```
**风险**: 如果生产环境未设置 `JWT_SECRET`，使用可预测的默认值会导致 Token 被伪造。
**修复**: 移除默认值，未设置时抛出错误阻止启动。

## 三、🟡 中等问题

### 4. B站搜索使用不一致的 axios 实例
**位置**: `server/src/services/chinaSearch.ts:181`
```typescript
const response = await axios.get<BilibiliSearchResponse>(...)
// 应该使用 noProxyAxios
```
**修复**: 统一使用 `noProxyAxios`。

### 5. 前端 `isLoading` 状态混淆
**位置**: `client/src/App.tsx:820,1501`
```typescript
// 第820行：应该检查 isDataLoading，但检查了 isLoading（来自 useAuth）
{isLoading ? (
  <div className="flex items-center justify-center py-16">...</div>
) : hotspots.length === 0 ? (
```
**修复**: 将 Dashboard 加载状态的 `isLoading` 改为 `isDataLoading`。

### 6. 微博 API 被封禁（403）
**位置**: `server/src/services/chinaSearch.ts:350`
```typescript
const response = await noProxyAxios.get('https://weibo.com/ajax/side/hotSearch', ...)
```
**问题**: 该接口现在需要登录态，返回 403 Forbidden。
**修复**: 增强错误处理，降级返回空数组；考虑替换数据源。

### 7. 国际搜索聚合未充分利用可用源
**位置**: `server/src/services/search.ts:343`
```typescript
export async function searchAll(query: string): Promise<SearchResult[]> {
  const results = await Promise.allSettled([
    searchBing(query),
    searchHackerNews(query)  // 只保留了2个源
  ]);
```
**问题**: 代码中有 `searchGoogle` 和 `searchDuckDuckGo` 的定义，但 `searchAll` 没有调用它们。
**修复**: 将 Google 和 DuckDuckGo 加回 `searchAll`（带容错）。

## 四、🟢 轻微问题

### 8. AI 调用缺少重试机制
**位置**: `server/src/services/ai.ts:33`
**问题**: 网络波动时 AI 分析会直接失败，没有指数退避重试。
**修复**: 添加带重试的 `callOpenRouter` 包装器。

### 9. 前端搜索按钮使用错误的 loading 状态
**位置**: `client/src/App.tsx:1501`
```typescript
<motion.button type="submit" disabled={isLoading}>
```
**修复**: 改为 `disabled={isDataLoading}`。

### 10. 部分代码使用 `any` 类型
**位置**: `server/src/services/ai.ts:9`, `client/src/App.tsx` 多处
**修复**: 逐步替换为具体类型（本次修复关键路径）。
